Ingeniería de Software

Los ingenieros de software se encargan de analizar, diseñar, crear y probar los sistemas informáticos y de software. Además, escriben programas de software para satisfacer las necesidades de un cliente o para resolver un problema particular.

Problemática

las computadoras no pueden manejar errores de sintaxis. un error de sintaxis detendrá la ejecución del programa.

Ventajas

es el arte del proceso por el cual se limpia, codifica, traza y protege el código fuente de programas computacionales, en otras palabras, es indicarle a la computadora lo que tiene que hacer. Un lenguaje de programación es el conjunto de instrucciones a través del cual los humanos interactúan con las computadoras. permiten a las computadoras procesar de forma rápida y eficientemente grandes y complejas cantidades de información.
